Integra o KLIP em qualquer coisa

O mesmo .exe é editor E bridge. Conduz por MCP, CLI, HTTP ou a tua própria chave. Copia e cola.

MCP

O .exe fala MCP (Claude Desktop / Code):

"klip": { "command": "KLIP Animator.exe", "args": ["--mcp-stdio"] }

HTTP bus

Porta em %TEMP%\klip_mcp.json:

curl -X POST http://127.0.0.1:PORT/call \
 -d '{"action":"insert_shape","params":{"shape":"squircle","fill":"#6D5EF6"}}'

BYOK

A tua chave Anthropic, local:

%APPDATA%\Klip\ai.json
{ "api_key": "sk-ant-..." }

CLI · SDK

Claude Code embutido; bus = HTTP em qualquer linguagem:

fetch(`http://127.0.0.1:${port}/manifest`).then(r=>r.json())
